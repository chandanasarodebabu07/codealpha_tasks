# AI Mini Projects Collection

This repository contains three AI-based projects developed using Python, Flask, Machine Learning, NLP, and Computer Vision.

---

# Projects Included

## 1. AI FAQ Chatbot

An intelligent chatbot that answers user queries using NLP preprocessing and cosine similarity.

### Features

- NLP preprocessing
- TF-IDF vectorization
- Cosine similarity matching
- Smart chatbot UI
- Browser-based interaction

### Technologies Used

- Python
- Flask
- NLTK
- Scikit-learn
- HTML
- CSS
- JavaScript

---

## 2. AI Language Translator

A real-time web-based language translation system.

### Features

- Multi-language translation
- Instant text translation
- Interactive UI
- Browser-based access

### Technologies Used

- Python
- Flask
- Googletrans
- HTML
- CSS
- JavaScript

---

## 3. AI Object Detection System

A real-time object detection system using webcam feed and YOLOv8.

### Features

- Real-time webcam detection
- Bounding box visualization
- Modern AI dashboard
- Browser-based live feed

### Technologies Used

- Python
- Flask
- OpenCV
- YOLOv8
- HTML
- CSS

# How These Projects Work

## FAQ Chatbot

1. Preprocesses text
2. Converts text into vectors
3. Uses cosine similarity
4. Returns best answer

---

## Language Translator

1. Accepts user text
2. Processes language translation
3. Displays translated output

---

## Object Detection

1. Captures webcam frames
2. Runs YOLOv8 detection
3. Draws bounding boxes
4. Streams results to browser
